# TANKZ Club: топик Tank 500, 2025-09

## Основные темы

**Лазерные фары Luban** — несколько поставщиков (Stopudova Ruslan в Алматы 550 тыс тг, Алёна @AlyonaAleshka в Костанае); отзывы положительные: намного ярче штатных.

**Прошивка АКПП 101** — Dmitriy Saparkin обновился в MyCar Кульджинка (первый в Алматы, 2025-09-03). Изменения: вернулся накат, расход по компу упал с 29.9 до 22; реальный ~26–28 л при средней 13 км/ч в пробках. Дилеры Астаны (Tank Nurly Zhol) о прошивке не знали.

**Расход топлива** — детальные данные по влиянию бензина 95/98/100; на 95 Прайм расход ниже обычного 95; при пробках (13 км/ч) 28 л — нормально для машин такого класса. Рекомендованные заправки: Гелиос, Royal Petrol, Lukoil, Gazprom.

**Подвеска** — Shaxi 2" установил @daursdt (первый отзыв); K-Man 2.5" для T500 официально подтверждено заводом, совместима с T400/T500, гарантия 12 мес/20 000 км; стойки стабилизатора OKING хватает ~20 000 км при еженедельном бездорожье.

**Колёса и запаска** — 265/65/18 влезает снизу без подрезки; 285/65/18 — после подрезки нижнего бампера; Maxxis Razr 275/70/18 LT (26 кг) + ET0 + расширители 3 см у Andrey Krassovskiy.

**Тормоза** — данные по пробегу смены колодок: 25–43 тыс км; скрипуны срабатывают при ~5 мм; диски: Монаер на Озоне, оригинал в ПДД ~45 тыс.

**Видеорегистраторы** — подключение к датчику дождя; несколько владельцев подтвердили простую установку.

**Бронирование кузова** — Авто Мажор (Алматы) рекомендован; после оклейки чек может гореть 1–2 дня.

**Адаптивный круиз в пробках** — отключается при остановке >2 сек; трогается медленно; мало пригоден для пробок.

**Новые аксессуары OFFROADART** — ДХО на крышу 150 тыс, анатомический руль 120 тыс, решётка 150 тыс, маркиза 70 тыс тг.

**Матрас надувной на передние сиденья** — 81 тыс тг + доставка 24 тыс = 105 тыс тг (Pindoduo); нужно опустить сиденья до нижней точки для ровной поверхности; подголовники снимаются (включая Sport).

**Прочее:** фаркоп — отметка в техпаспорте не нужна; обогрев лобового — экран → климат → правый нижний угол; подсветка мигает = сигнал о машине в мёртвой зоне (не глюк); омыватель — ложное оповещение проходит само за 3 дня; масло в мостах при обкатке — замена на 7–10 тыс км; знакомый с T500 Hi4-T — фары у гибрида другие.
