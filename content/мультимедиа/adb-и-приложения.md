# ADB-доступ к головному устройству

Головное устройство Tank 500 работает на Android. Через ADB (Android Debug Bridge) можно устанавливать сторонние APK-приложения и выполнять команды в терминале ГУ.

## Требования

- **Полное инженерное меню** — стандартное урезанное инженерное меню не подходит. Открыть инженерное меню **без специального оборудования нельзя** — нужен диагностический лаунчер (например, Autel) / OBD-сканер; дилеры и специалисты с лаунчером могут открыть доступ (Руслан @ORO1986, 2025-02-18).
- Windows-ноутбук с установленным ADB (Android Platform Tools).

## Подключение по Wi-Fi (рекомендуется)

1. В инженерном меню включить переключатель **"Network ADB mode port switch"**.
2. Убедиться, что ноутбук и ГУ в одной Wi-Fi сети.
3. Подключиться через `adb connect [IP-адрес ГУ]:5555`.
4. На экране ГУ появится окно с китайскими символами — нажать **OK** (разрешить подключение).

## Подключение по USB

1. В инженерном меню перевести **"USB mode switching"** в положение **"peripheral"**.
2. Подключить USB-кабель к ГУ.

> Оба параметра (`Network ADB mode port switch` и `USB mode switching`) **сбрасываются в исходное положение** после перезагрузки головного устройства — нужно включать заново при каждой сессии.

*Инженерное меню: выделены "Network ADB mode port switch" и "USB mode switching" (Dauren KRG ADK, 2024-02)*

## Команды

```
adb shell              # терминал ГУ
adb install app.apk    # установить APK
adb uninstall com.pkg  # удалить приложение
```

Установленные приложения отображаются в инженерном меню → раздел **"installed apps"**.

## Ограничения (на февраль 2024)

- Установить APK через `adb install` можно — приложение появляется в инженерном меню.
- **Запустить установленное приложение из интерфейса ГУ не удалось** — нет точки входа в стандартном UI. Вопрос остаётся открытым.

## GWM Split — альтернативный вид приборной панели

На T500 можно поставить альтернативный вид приборки — **GWM Split**. Это не отдельный гаджет, а установка приложения через инженерное меню (ADB + лаунчер). Split выводит дополнительные показания (расход, температуры агрегатов, обороты) поверх основного экрана ГУ; работает при открытом инженерном меню (Вячеслав Л, 2025-02-19).

**Что нужно:** лаунчер + ноутбук + USB-USB кабель (Руслан @ORO1986, Актау, 2024-12).

**Бонус:** после установки кнопка голосового ассистента на руле (которая у большинства КЗ-машин является заглушкой) начинает работать при подключении CarPlay (Dmitriy Saparkin, 2024-12-25).

Телеграм-канал с инструкциями: **t.me/t500_split** (Вячеслав Л, 2024-12-24).

В КЗ установку делает @ORO1986 (Руслан, Актау) при личном визите — удалённо не работает. → [[актау|Актау]]
