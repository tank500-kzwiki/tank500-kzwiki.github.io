# Tank 500

> Неофициальная вики сообщества. См. [[о-вики|О проекте и правовая информация]] — отказ от ответственности и удаление данных по запросу.

Полноразмерный рамный внедорожник GWM. Основная модель Tank, официально продаваемая в Казахстане через дилерскую сеть Astana Motors.

## Технические характеристики

### Двигатель и трансмиссия

| Параметр | Значение |
|----------|----------|
| Тип двигателя | Бензиновый, комбинированный впрыск, двойной турбонаддув |
| Обозначение | 3.0T V6 (E30Z) |
| Рабочий объём | 2993 см³ |
| Конфигурация | 6 цилиндров, V-образное |
| Мощность (Premium / Tech Plus) | 299 л.с. / 6000 об/мин |
| Мощность (Black) | 354 л.с. / 6000 об/мин |
| Крутящий момент | 500 Нм / 1500–4000 об/мин |
| Коробка передач | Автоматическая, 9-ступенчатая, гидромеханическая (9AT) |
| Тип привода | Полный, TOD (многодисковая муфта в приводе передних колёс) |
| Понижающая передача | Есть |

> Значения мощности — официальные (тех. паспорт / прайс-лист). Фактическая мощность КЗ-версии обсуждается владельцами: на стенде Dynomax КЗ-сток показал 310–316 л.с., а заявленные для Black/Black Trail 354 л.с. оспариваются (в сообществе фигурируют ~309 л.с.). Подробнее → [[чип-тюнинг|Чип-тюнинг двигателя]].

### Обкатка

Первые **1000 км**: не превышать 100 км/ч, избегать резких ускорений и экстренных торможений.

### 48В мягкая гибридная система

Все комплектации Tank 500 3.0T оснащены 48-вольтовой лёгкой гибридной системой. Индикатор **READY** на приборной панели сигнализирует о готовности системы. Отвечает за рекуперацию энергии при торможении и сглаживание работы двигателя.

## Отличия комплектаций

### Premium → Tech Plus (добавляется)

**Управление:**
- Блокировка переднего межколёсного дифференциала

**Экстерьер:**
- Боковые подножки с электроприводом (вместо фиксированных)

**Интерьер:**
- Отделка элементов алькантарой (вместо ткани)
- Выдвижные шторки на стёклах задних дверей
- Доводчик двери багажника

**Сиденья:**
- Кожа NAPPA (вместо искусственной кожи)
- Память положений водительского сиденья + функция Welcome
- Электрорегулировка подголовника переднего пассажира в 4-х направлениях
- Электрорегулировка сиденья переднего пассажира в 6-ти направлениях
- Массаж и вентиляция сидений переднего ряда

**Безопасность:**
- Предупреждение для безопасного открытия двери (DOW)
- Система помощи при выезде с парковки задним ходом (RCTA) с торможением (RCTB)
- Экстренное удержание в полосе (ELK)
- «Умное уклонение» (Smart Dodge)
- Предупреждение о наезде сзади (RCW)

**Комфорт:**
- Электропривод регулировки рулевой колонки
- Фары с автоматическим корректором (ALS) и функцией приветствия
- Система активного шумоподавления
- Шумоизолированное остекление задних дверей
- Панорамная крыша
- Зеркала с подсветкой зоны посадки и памятью положения
- Система контроля качества воздуха (фильтр N95 + ароматизация)

**Мультимедиа:**
- Премиальная акустика Infinity (12 динамиков) + сабвуфер (вместо 8 + сабвуфер)
- Беспроводная зарядка для пассажиров заднего ряда

### Tech Plus → Black (добавляется)

- Дизайн-пакет BLACK: антихром + запасное колесо на задней двери
- Увеличенная мощность: 354 л.с. (вместо 299)
- Дополнительный топливный бак 50 л
- Ускоренный разгон: 8,8 с (вместо 9,6)

## Стандартное оборудование (все комплектации)

### Управление
- Рулевая колонка, регулируемая по высоте и вылету
- Электроусилитель с переменным усилием и выбором режима
- Блокировка заднего межколёсного дифференциала
- Система старт/стоп
- Помощь при спуске и при трогании на подъёме
- Система выбора режима движения с режимом «Эксперт»
- Поддержание малой скорости на бездорожье (Creep Mode)
- Помощь при повороте на бездорожье (Tank Turn)
- Система автоматической парковки

### Экстерьер
- Легкосплавные диски 19"
- Распашная дверь багажника
- Рейлинги на крыше
- Люк на крыше
- Подготовка под установку ТСУ

### Интерьер
- Мультифункциональное рулевое колесо с подогревом
- Подрулевые лепестки переключения передач
- Цифровая приборная панель 12,3"
- Контурная динамическая подсветка интерьера
- Розетка 220V
- Шторка багажника

### Сиденья
- Подогрев сидений переднего и заднего ряда
- Сиденье водителя с электрорегулировкой в 6-ти направлениях
- Электрорегулировка поясничной поддержки водителя
- Регулировка угла наклона спинки заднего ряда
- Складываемый задний ряд 60:40
- Центральный подлокотник заднего ряда с подстаканниками
- Сенсорный экран управления функциями в подлокотнике (только 5-местные Tech Plus / Black)

### Мультимедиа
- Мультимедийная система с 14,6" сенсорным дисплеем
- Аудиосистема с AM/FM и Bluetooth
- USB спереди и сзади
- Разъём для видеорегистратора
- Беспроводная зарядка для переднего ряда
- Беспроводное подключение Apple CarPlay / Android Auto

### Безопасность
- ESP + TCS + RMI
- Помощь при экстренном торможении (BAS)
- Автоматическое торможение на малой скорости
- AEB с FCW + распознавание пешеходов и велосипедистов
- Ограничитель скорости
- Адаптивный круиз-контроль (ACC) с функцией движения на малых скоростях
- Распознавание дорожных знаков (TSR)
- Интеллектуальный круиз-ассистент (ICA)
- LDW + LKA + LCK (удержание в полосе)
- AEB Crossroad (предотвращение столкновений на перекрёстках)
- Камера кругового обзора с «прозрачным» капотом
- Передние и задние датчики парковки
- Контроль усталости водителя
- 6 подушек безопасности (фронтальные + боковые + шторки)
- Преднатяжители ремней переднего и заднего ряда
- Крепления детских кресел (ISOFIX)

### Комфорт
- Бесключевой доступ и запуск кнопкой
- Трёхзонный климат-контроль
- Проекция на лобовое стекло (HUD)
- Электрообогрев лобового стекла и форсунок омывателя
- Шумоизоляционное остекление спереди
- Светодиодные фары ближнего и дальнего света
- Автоматическое управление дальним светом
- Передние ПТФ и задние динамические фонари
- Подсветка поворота
- Датчик света и дождя
