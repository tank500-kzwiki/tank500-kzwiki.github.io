# Аббревиатуры и сленг

Расшифровка технических аббревиатур и сленга, встречающихся в вики и в чате владельцев Tank 500. Привязанные к конкретному варианту факты см. на профильных страницах (ссылки в строках).

## Варианты и исполнения Tank 500

| Термин | Значение |
|--------|----------|
| 3.0T | Бензиновый Tank 500 с двигателем 3.0T V6 (E30Z), 9AT, привод TOD — основной вариант в КЗ |
| V6 | 1) Конфигурация двигателя (6 цилиндров); 2) в контексте коробки — версия прошивки АКПП (см. ниже) |
| E30Z | Заводское обозначение двигателя 3.0T V6 |
| Hi4 / Hi4-Z | Подключаемый гибрид (PHEV) Tank 500; в КЗ официально не продаётся ([[гибридная-версия|подробнее]]) |
| Hi4-T | Гибридная силовая установка (роднится с Tank 400); для КЗ — слухи ([[рестайлинг|рестайлинг]]) |
| PHEV | Plug-in Hybrid — подключаемый гибрид (заряжается от сети) |
| BT | Black Trail — исполнение с доп. баком и иной топливной системой ([[троение-bt|нюансы BT]]) |
| Black Edition | Топовое исполнение (антихром, запаска на двери, заявлено 354 л.с.) |
| Grey Edition | Стандартное/«серое» исполнение (~290–299 л.с.) |

## Трансмиссия и привод

| Термин | Значение |
|--------|----------|
| 9AT | 9-ступенчатая гидромеханическая АКПП |
| АКПП / КПП | Автоматическая / коробка переключения передач |
| TOD | Torque On Demand — полный привод с многодисковой муфтой ([[полный-привод|режимы]]) |
| 4H / 4L | Полный привод: высокий / низкий (понижающий) ряд раздатки |
| МГ | Мягкий гибрид (48В) — влияет на характер пинков и задержек АКПП ([[48в-батарея|48В батарея]], [[48в-батарея-неисправности|неисправности]]) |
| BSG / ISG | Стартер-генераторы 48В системы; добавляют момент при разгоне |
| V3 / V4 / V6 / V7 | Версии прошивки АКПП; V6 устраняет пинки у большинства ([[пинки-коробки|пинки коробки]]) |

## Мультимедиа и электроника

| Термин | Значение |
|--------|----------|
| ГУ | Головное устройство (мультимедийный блок) ([[зависание-гу|зависание ГУ]]) |
| OTA | Обновление прошивки «по воздуху» |
| ADB | Android Debug Bridge — доступ к ГУ для установки приложений ([[adb-и-приложения|ADB-доступ]]) |
| БК | Бортовой компьютер (показания расхода и др.) |
| TBox | Телематический блок; в КЗ сервис не работает ([[tank-connect|Tank Connect]]) |
| HUD | Проекция на лобовое стекло |

## Системы помощи водителю (ADAS)

| Термин | Значение |
|--------|----------|
| LKA | Удержание в полосе; в T500 включается при каждом запуске ([[удержание-полосы|подробнее]]) |
| LDW | Предупреждение о выходе из полосы |
| LCK / ELK | Центрирование в полосе / экстренное удержание в полосе |
| ACC | Адаптивный круиз-контроль ([[круиз-контроль|круиз-контроль]]) |
| ICA | Интеллектуальный круиз-ассистент |
| TSR | Распознавание дорожных знаков |
| AEB / FCW | Автоторможение / предупреждение о фронтальном столкновении |
| RCTA / RCTB / RCW | Помощь при выезде задним ходом / с торможением / предупреждение о наезде сзади |
| DOW | Предупреждение безопасного открытия двери |
| BSM | Контроль слепых зон |
| ALS / AFS | Автокорректор фар / адаптивная система головного света |
