# Скачать вики и пользоваться с LLM

Вся вики — это обычные markdown-файлы. Их можно скачать и пользоваться **локально с любой LLM**
(например, Claude Code, ChatGPT, локальные модели): задавать вопросы по Tank 500, а модель будет
отвечать на основе скачанных страниц.

## Где скачать

Исходники вики и инструкции для LLM лежат в публичном репозитории на GitHub:

**https://github.com/tank500-kzwiki/tank500-kzwiki.github.io**

Скачать целиком можно двумя способами:

- **Кнопка «Code → Download ZIP»** на GitHub — без установки чего-либо.
- **`git clone`** (если есть git):
  ```
  git clone https://github.com/tank500-kzwiki/tank500-kzwiki.github.io.git
  ```

## Что внутри

- **`content/`** — все страницы вики в формате markdown (то, что вы видите на сайте), вместе с
  изображениями в `content/assets/`.
- **`llm-kit/`** — материалы для работы с LLM:
  - `llm-kit/CLAUDE.md` — описание структуры вики и правил (как LLM должна искать и отвечать);
  - `llm-kit/skills/` — операции по ведению вики (разбор источников, проверка целостности).

> В публикацию **не входят** сырые источники (`raw/`) — только готовые страницы вики и инструкции.

## Как пользоваться с LLM

1. Скачайте репозиторий (см. выше).
2. Откройте папку в инструменте с LLM (например, **Claude Code** в каталоге репозитория) — модель
   увидит `llm-kit/CLAUDE.md` и страницы в `content/`.
3. Или вручную дайте модели файл `CLAUDE.md` и нужные страницы из `content/` и задавайте вопросы.
4. Просите модель **отвечать на основе вики** и ссылаться на страницы — так ответы будут точными.

Отдельную страницу можно скачать прямо из репозитория: откройте нужный `.md` в `content/` на GitHub
и нажмите кнопку загрузки (Raw → сохранить).

## Рекомендуемый способ: Obsidian + LLM

Самый удобный вариант для постоянного использования — открыть скачанную папку `content/` как
**хранилище [Obsidian](https://obsidian.md)** (бесплатное приложение для заметок). Вики изначально
написана в формате Obsidian, поэтому всё работает «из коробки»:

- **Навигация как на сайте** — перекрёстные ссылки `[[…]]`, граф связей, поиск по всем страницам,
  просмотр изображений. По сути вы получаете ту же вики, но локально и без интернета.
- **Связка с LLM** — держите файлы в Obsidian-хранилище, а рядом подключите LLM (например, запустите
  **Claude Code** в той же папке, или используйте плагины Obsidian для ИИ). Тогда можно и читать/
  редактировать заметки вручную, и спрашивать модель — она отвечает на основе локальных страниц.

Как настроить:

1. Скачайте репозиторий (см. «Где скачать»).
2. Установите Obsidian → **Open folder as vault** → выберите папку `content/` (или весь репозиторий,
   если хотите видеть и `llm-kit/`).
3. Для работы с ИИ запустите LLM-инструмент в этой же папке и дайте модели `llm-kit/CLAUDE.md` как
   инструкцию — дальше задавайте вопросы по Tank 500.

> Это та же конфигурация, в которой ведётся вики: файлы в Obsidian-хранилище + LLM-обвязка для
> поиска и ответов. Вы получаете и личную базу знаний, и ИИ-помощника по машине одновременно.
