# Скачать вики и пользоваться с LLM

Вся вики — это обычные markdown-файлы. Можно задавать вопросы по Tank 500 **любой LLM**
(например, claude.ai, ChatGPT, Claude Code, локальные модели), и модель будет отвечать на основе
страниц вики. Для разовых вопросов ничего скачивать не нужно (см. раздел **«Самый простой способ»** ниже);
для постоянного использования вики можно скачать целиком.

> **Основной источник вики** — архивы сообщества **[TANKZ Club](https://tank.com.kz/ru/tankz-club/)**
> (клуб владельцев Tank в Казахстане). Подробнее об источниках и правах — на странице
> [[о-вики|О проекте и правовая информация]].

## Где скачать

Исходники вики и инструкции для LLM лежат в публичном репозитории на GitHub:

**https://github.com/tank500-kzwiki/tank500-kzwiki.github.io**

Скачать целиком можно двумя способами:

- **Кнопка «Code → Download ZIP»** на GitHub — без установки чего-либо.
- **`git clone`** (если есть git):
  ```
  git clone https://github.com/tank500-kzwiki/tank500-kzwiki.github.io.git
  ```

## Что внутри

- **`content/`** — все страницы вики в формате markdown (то, что вы видите на сайте), вместе с
  изображениями в `content/assets/`.
- **`llm-kit/`** — материалы для работы с LLM:
  - `llm-kit/CLAUDE.md` — **ключевой файл-инструкция**. Именно он заставляет модель отвечать
    «вики-first»: сначала прочитать `content/index.md`, найти релевантные страницы, ответить
    **на основе вики** и поставить ссылки на страницы-источники, а если в вики ответа нет — явно
    это пометить. Дайте этот файл модели — и она будет отвечать по вики, а не «из головы».
  - `llm-kit/skills/` — операции по ведению вики (разбор источников, проверка целостности).

> В публикацию **не входят** сырые источники (`raw/`) — только готовые страницы вики и инструкции.

## Самый простой способ — без установки

Ничего скачивать и устанавливать не нужно. Достаточно браузера и любого чат-ИИ
(**[claude.ai](https://claude.ai)**, **ChatGPT** и т.п.):

1. Откройте чат и **приложите два файла**: `llm-kit/CLAUDE.md` (инструкция, см. «Что внутри»)
   и ту страницу вики, по которой хотите спросить.
2. Одну страницу удобно взять прямо из репозитория: откройте нужный `.md` в `content/` на
   GitHub и нажмите **Raw → сохранить** (или скопируйте текст в чат).
3. Задайте вопрос про Tank 500 и **попросите отвечать на основе вики и ссылаться на страницы** —
   так ответы будут точными, а не выдуманными.

Если не знаете, какая страница нужна — приложите ещё `content/index.md` (каталог вики) и
попросите модель сначала найти по нему подходящие страницы.

## Как пользоваться с LLM (полная локальная связка)

Способы ниже дают доступ ко **всей вики сразу** (модель сама ходит по страницам и ссылкам) —
удобно для постоянного использования, но требуют установки инструментов.

1. Скачайте репозиторий (см. «Где скачать»).
2. Откройте папку в инструменте с LLM (например, **Claude Code** в каталоге репозитория) — модель
   увидит `llm-kit/CLAUDE.md` и все страницы в `content/` и сможет сама искать ответ по вики.
3. Просите модель **отвечать на основе вики** и ссылаться на страницы — так ответы будут точными.

## Рекомендуемый способ для постоянного использования: Obsidian + LLM

Самый удобный вариант для постоянного использования — открыть скачанную папку `content/` как
**хранилище [Obsidian](https://obsidian.md)** (бесплатное приложение для заметок). Вики изначально
написана в формате Obsidian, поэтому всё работает «из коробки»:

- **Навигация как на сайте** — перекрёстные ссылки `[[…]]`, граф связей, поиск по всем страницам,
  просмотр изображений. По сути вы получаете ту же вики, но локально и без интернета.
- **Связка с LLM** — держите файлы в Obsidian-хранилище, а рядом подключите LLM (например, запустите
  **Claude Code** в той же папке, или используйте плагины Obsidian для ИИ). Тогда можно и читать/
  редактировать заметки вручную, и спрашивать модель — она отвечает на основе локальных страниц.

Как настроить:

1. Скачайте репозиторий (см. «Где скачать»).
2. Установите Obsidian → **Open folder as vault** → выберите папку `content/` (или весь репозиторий,
   если хотите видеть и `llm-kit/`).
3. Для работы с ИИ запустите LLM-инструмент в этой же папке и дайте модели `llm-kit/CLAUDE.md` как
   инструкцию — дальше задавайте вопросы по Tank 500.

> Это та же конфигурация, в которой ведётся вики: файлы в Obsidian-хранилище + LLM-обвязка для
> поиска и ответов. Вы получаете и личную базу знаний, и ИИ-помощника по машине одновременно.
